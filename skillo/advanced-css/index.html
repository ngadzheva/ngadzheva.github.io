CSS трансформации и анимации

Градиент

CSS градиентът позволява да визуализираме плавен преход между 2 или повече цвята.

CSS дефинира 3 вида градиенти:

  • линеен градиент - надолу, нагоре, наляво, надясно или по диагонал
  • радиален градиент - дефиниран от централна точка
  • коничен градиент - върти се около централна точка

Линеен градиент

background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

Радиален градиент

background-image: radial-gradient(shape size at position, start-color, ..., last-color);

Коничен градиент

background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...);

Сянка

Със CSS можем да слагаме сянка на текст и на елементи

  • text-shadow: x-offset y-offset blur color
  • box-shadow: x-offset y-offset blur spread color inset. Може да се изредят няколко чрез запетайка, подобно на background.

Трансформации

2D трансформации

transform: function();
  • translate - премества елемент по хоризонтала и/или вертикала
  • rotate - завърта елемент по часовниковата срелка или обратно на часовниковата стрелка
  • translate - увеличава или намалява размера на елемент
  • skew - накланя елемент по хоризонтала и/или вертикала

Преходи

CSS преходите дават възможност да променим плавно някакви стилове на елементи.

За да направим такъв плавен преход, трябва да определим 2 неща:

  • стила, който искаме да променим
  • за какво време искаме да се случи промяната
  • transition-delay - определя след колко време ще започне прехода в секунди
  • transition-duration - определя времето за извършване на прехода в секунди
  • transition-property - определя стила, чиято стойност ще се сменя
  • transition-timing-function - определя скоростта на изпълнение на прехода
  • transition - определя горните 4 наведнъж

Анимации

Със CSS можем да анимираме HTML елементи, без да се налага да използваме за целта JavaScript.

За да направим анимация, трябва да дефинираме кадри за извършването на анимацията.

Кадрите определят в какъв момент от времето какви стилове на анимирания елемент ще се променят и как.

Дефиниране на кадри

							
                /* The animation code */
                @keyframes example {
                    from {background-color: red;}
                    to {background-color: yellow;}
                }
                
                /* The element to apply the animation to */
                div {
                    width: 100px;
                    height: 100px;
                    background-color: red;
                    animation-name: example;
                    animation-duration: 4s;
                }

Забавяне на анимация

							
                        div {
                            width: 100px;
                            height: 100px;
                            position: relative;
                            background-color: red;
                            animation-name: example;
                            animation-duration: 4s;
                            animation-delay: 2s;
                        }

Определяне колко пъти ще се изпълнпва анимацията

							
                        div {
                            width: 100px;
                            height: 100px;
                            position: relative;
                            background-color: red;
                            animation-name: example;
                            animation-duration: 4s;
                            animation-iteration-count: 3;
                        }

Определяне на скоростта на анимацията

							
            #div1 {animation-timing-function: linear;}
            #div2 {animation-timing-function: ease;}
            #div3 {animation-timing-function: ease-in;}
            #div4 {animation-timing-function: ease-out;}
            #div5 {animation-timing-function: ease-in-out;}
