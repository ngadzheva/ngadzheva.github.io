Responsive дизайн

Какво е responsive дизайн?

Responsive дизайнът помага страниците ни да изглеждат добре на различни размери екрани.

Какво е viewport?

Видимата за потребителя част от уеб страницата. Големината зависи от екрана на устройството, на което е отворена страницата.

Правила за разработване на responsive дизайн

  • Разработете дизайна първо за мобилни устройства.
  • Не разчитайте само на един конкретен размер на viewport.
  • Избягвайте фиксирани размери на елементите.
  • Задайте min-width или max-width на елементите и ги комбинирайте с "плаващ" размер.
  • Използвайте media query за дефиниране на различна стилизация за големи и малки екрани.
  • Използвайте модерни техники за създаване на layout на вашите страници като flex и grid.
  • Променяйте layout-a и размерите на елементите в зависимост от големината на екрана.
  • Приоритизирайте и скрийте съдържанието в зависимост от контекста на потребителя.

Media Queries

                            
                    @media not|only mediatype and (expressions) {
                        CSS-Code;
                    }

Дават възможност да определяме различни стилове на елементите за различни видове медия.

Те могат да проверяват много неща като:

  • ширина и височина на viewport-a
  • ширина и височина на устройството
  • ориентация на екрана на устройството
  • резолюция на екрана

Определяне на breakpoints

Breakpoint-ите определят при какъв размер на екрана да се променят стиловете на страницата.

                            
                    .col {
                        width: 100%;
                    }

                    @media only screen and (min-width: 768px) {
                        .col {
                            width: 50%;
                        }
                    }

Стандартни breakpoint-и

                            
                /* Extra small devices (phones, 600px and down) */
                @media only screen and (max-width: 600px) {...}
                
                /* Small devices (portrait tablets and large phones, 600px and up) */
                @media only screen and (min-width: 600px) {...}
                
                /* Medium devices (landscape tablets, 768px and up) */
                @media only screen and (min-width: 768px) {...}
                
                /* Large devices (laptops/desktops, 992px and up) */
                @media only screen and (min-width: 992px) {...}
                
                /* Extra large devices (large laptops and desktops, 1200px and up) */
                @media only screen and (min-width: 1200px) {...}

Засичане на ориентацията на екрана на устройството

С media query можем да променим layout-a на страницата ни в зависимост от ориентацията на екрана на устройството.

                            
                @media only screen and (orientation: landscape) {
                    body {
                        background-color: lightblue;
                    }
                }

Видове layout-и

  • Multicolumn - колони за текст, не се държи добре на мобилни устройства
  • Flexbox - предоставя доста флексибилност в подреждането на елементи в контейнер едномерно
  • Grid - най-често използваният layout за подреждане на елементи в 2D пространството
